# Market Structure Notes

## Volatility Patterns
- Monday gaps often fill by Wednesday
- End-of-month rebalancing creates volume spikes
- News events cluster at 8:30 AM EST

## Key Metrics to Track
| Metric | Why It Matters | Data Source |
|--------|---------------|-------------|
| VIX | Fear gauge | CBOE |
| OI (Open Interest) | Sentiment | Exchange API |
| Funding rates | Leverage levels | Perp exchanges |
| Order book depth | Liquidity | Exchange API |

## Analysis Framework
1. Macro trend (weekly)
2. Volatility regime (daily)
3. Micro-structure (hourly)
4. Entry timing (minute)
